# GovnoScript

GovnoScript - простой скриптовый язык для автоматизации управления клавиатурой и мышью.

Установка:
```commandline
pip install govnoscript
```

## Синтаксис

Код пишется либо в `.govno`-файле, либо передается в Python напрямую при запуске.

Каждая команда находится на отдельной строке:

### Управление мышью
`MOVETO x y time` - передвинуть курсор мыши на `(x;y)` за `time` секунд.  
`CLICK mouse_button` - нажать одну из следующих кнопок мыши: `LEFT`, `MIDDLE`, `RIGHT`.  

### Управление клавиатурой
`WRITE time Some Text` - напечатать "Some Text" с задержкой `time` секунд между клавишами 
(`0` - чтобы напечатать моментально).  
`WRITE_BY_KEYBOARD time Some Text` - аналог `WRITE`, имитирующий нажатие и отпускание клавиш.  
`PRESS button` - зажать кнопку на клавиатуре.  
`RELEASE button` - отпустить кнопку на клавиатуре.  
`TAP button` - объединяет `PRESS` и `RELEASE` в одной команде.  
`HOTKEY btn1+btn2+...` - нажать сочетание клавиш. Клавиши разделяются через `+`.  

### Другие команды

`WAIT time` - ожидание `time` секунд.  
`TIME` - вывести в терминал время выполнения скрипта в миллисекундах. Можно использовать сколько угодно раз в любом месте
скрипта.

### Переменные

В GovnoScript есть поддержка переменных. В них можно записать шаблонные значения и вынести их в начало скрипта, чтобы
избавится от поиска строго вбитых выражений в больших скриптах. Переменные задаются следующим образом:
```
some_var = variable value
```
Чтобы получить значение `variable value`, нужно прописать `%some_var%`:
```
WRITE 0 %some_var%
```
Данная команда моментально напечатает `variable value`.

Переменная должна быть указана ДО момента её использования. Название переменной не должно содержать специальных символов, а также пробелов. Значение может содержать в себе
любое число или текст.

### Написание чисел

В GovnoScript время указывается в секундах. Также при вызове команды или при записи в переменную можно указать дробное
число:
```
WAIT 2.5
WAIT 7,3
```
В качестве разделителя дробной и целой части поддерживаются как точка, так и запятая.

Также при указании числа можно задать ему погрешность. Это будет означать, что при выполнении скрипта будет выбрано
случайное число, входящее в указанный диапазон. Число с погрешностью задается через `число~погрешность`:
```
WRITE 1~0.5 Hello World!
```
Эта строка распечатает "Hello World", притом время между нажатиями клавиш будет в диапазоне от `0.5` до `1.5` - оно определится
случайным образом.

## Комментарии и стиль написания кода

Также в коде можно оставлять комментарии. Для этого строка должна начинаться с `#` и после одного пробела должен быть указан сам текст комментария:
```
# Это комментарий

write 0 Hello World
```

Также для большей читаемости можно оставлять пустые строки, а также указывать имена команд в нижнем регистре


## Запуск

На данный момент есть два способа запуска GovnoScript:
```python
import govnoscript

govnoscript.run("file.govno") # Запустить .govno-файл
govnoscript.interpret([
    "WAIT 5",
    "WRITE 0.5~0.25 Hello World!" 
]) # Запускает сырой код из массива строк
```

Также при каждом из данных видов запуска можно указать новые переменные:
```python
import govnoscript

govnoscript.run("file.govno", script_vars={
    "some_var": "variable value"
})
```
